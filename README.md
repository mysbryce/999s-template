### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mysbryce/999s-template.git
   ```
2. Install NPM packages
   ```sh
   bun install
   ```
3. Development Script
   ```sh
   bun dev
   ```
4. Development UI
    ```sh
    cd web
    bun dev
    ```
